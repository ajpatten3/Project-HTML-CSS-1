Lesson 3 - Adding CSS and HTML

CSS Style

CSS style is the color or theme of the blocks in the HTML text. CSS stands for cascading style sheet. Cascading style sheet allows the programmer to change the color and theme of different elements by giving them a "class". By using "class" this gives the elements it's associated with the specified color or theme.

Avoiding Repetition

Avoiding problems: Keeping consistent code is very important to becoming a good programmer. This concept is important because if you don't keep your code consistent then you end up making the same mistake over and over. This can be hard to fix as well since you have to look at every line of code for a different error. Consistency is important for any new concept in order to maintain good fundamental skills. Consistency is used in programming with closing tags and end tags. These are very important to remember if the code is to be proper format. This concept also comes into play with spacing. The issue I ran into was line spacing with the I.D. tag and linking. These are easy problems to fix but are important concepts to remember going forward.
